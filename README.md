# Photo 2 Comics Bot
Check out how a photo would've looked in a comics book!

Give it a try here: 
https://t.me/photo2comicsbot

# Install 
npm install 
* set up your ecosystem.config.js for pm2 env variables, or use other preferred method

# Run
* npm run tsc
* pm2 start ecosystem.config.js
